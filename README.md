## Sobre o projeto
Aplicação desenvolvida para consolidar conceitos avançados de React, focando em uma arquitetura profissional de componentes e uma interface de alta fidelidade baseada em um ambiente real de produção.

## Tecnologias utilizadas
* **React.js + Vite**
* **Tailwind CSS**
* **React Router v6**
* **Axios (Consumo de API REST)**


  ## Diferenciais Técnicos
* **Arquitetura Modular:** Código focado em Clean Code, sem lógica no arquivo principal; tudo foi segmentado em componentes reutilizáveis e pastas organizadas.
* **Fidelidade Visual (UX/UI):** Reprodução fiel da Home de um sistema real para testar limites de design, usabilidade e fidelidade visual.
* **Componentização:** Criação de componentes reutilizáveis para garantir um código limpo.
* **Escalabilidade**: Estrutura de pastas pensada para manutenção facilitada e crescimento do projeto.
